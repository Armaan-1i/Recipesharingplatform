RecipeWorld - Frontend demo
Features:
- Dark mode toggle (stored in localStorage)
- Save recipes locally (localStorage)
- Search and filter
- Recipe details page (static, populated from localStorage)
- Export / Import JSON of recipes

How to run:
1. Unzip recipe_frontend.zip
2. Keep folders 'html' and 'css' and 'js' as siblings in the project folder
3. Open html/index.html in a browser or use Live Server (recommended)
